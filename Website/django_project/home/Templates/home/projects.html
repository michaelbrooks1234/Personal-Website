    NASA picture of the day app.

    This was my first actual project that I had undertaken. It is a GUI app made in python with tkinter.
    It takes the date from the input box and sends a request to the NASA API with that date and then recieves a json
    with information about the image of the day for that date. I decided to show only a description of the image, and the actual image itself.

    Simple Calculator

    This is my second real app that I made. It is a calculator that can perform basic maths like addition, subtraction, multiplication, division,
    and some other somewhat useful opperations. I made it in c++ with QT. It took some time because of all the buttons, but implementing the math wasn't hard
    and I used two different integer variables to store the first or second number that was inputted. Then I just used a function to calculate what was needed.
    and set the answer to the first number so that calculations could be done on the new number without having to type it out again.
